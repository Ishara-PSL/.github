# 🤟 Palestinian Sign Language Learning Platform (Ishara-PSL)

An AI-powered educational platform for learning Palestinian Sign Language (PSL) through interactive lessons, multimedia content, quizzes, and real-time gesture evaluation using Computer Vision.

---

##  About the Project

Communication with deaf and hard-of-hearing individuals remains a challenge due to the limited availability of Palestinian Sign Language (PSL) learning resources.

This project aims to build the first AI-assisted platform dedicated to teaching Palestinian Sign Language through an engaging and interactive learning experience.

Inspired by modern language-learning applications such as Duolingo, the platform combines structured lessons with Artificial Intelligence to help learners practice signs and receive instant feedback.

---

##  Project Objectives

- Develop a structured Palestinian Sign Language curriculum.
- Create interactive learning lessons.
- Provide high-quality video demonstrations.
- Support camera-based sign practice.
- Evaluate user gestures using AI.
- Track learner progress.
- Promote accessibility and social inclusion.
